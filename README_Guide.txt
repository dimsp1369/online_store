ONLINE STORE GUIDE

INTRO
BackEnd

С чем работаем?
1. Node js
2. Express - framework
3. Postgre SQL - система управления базами данных
4. Sequelizer - ORM - для реляционных баз данных на node js. ORM - технология позволяющая связывать код с базами данных (т.е. не пишем напрямую SQL запрос, а вызываем функицю которая работает с БДю

Что сделаем?

1. Построим диаграмму базы данных, состоящей из 8 таблиц.
2. Реализуем авторизацию полизователя по JWT токену ( JSONWebToken )
3. Полноценное REST API
4. Поработаем с ролями.

FrontEnd
С чем работаем?
1. react js
2. React bootstrap - графика
3. Axios - запросы к серверу
4. React-router-dom - навигация
5. MobX - стейт менеджмент

Step 1. Work with backend

npm install express pg pg-hstore sequelize cors dotenv  - (
pg - postgreSQL
pg-hstore - пакет узлов
dotenv - модуль переменных окружения
)

Устанавливаем Nodemone ( npm i -D nodemon ) для автоматической перезагрузки сервера

Устанавливаем express-fileupload (npm i express-fileupload) для загрузки файлов
